# NextJs 12 boilerplate

## System Requirement

- Build by Nextjs
- Nodejs v14.15.0
- TypeScript
- ReactJS
- Redux
- RxJS
- Observable
- Jest

## Development

```
docker-compose up
```

or

```
yarn dev
```

## Production

```
yarn run build
```

```
yarn start
```

## TEST

```
yarn test
```
